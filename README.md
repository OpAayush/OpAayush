# Aayush

Founder and Software Engineer

Self-taught, Fast learner, building distributed systems, APIs, and web platforms from first principles. I lean toward unconventional solutions when they produce simpler architecture, better performance, or a more intuitive experience. Most of what I build starts as a whiteboard problem before it becomes a stack decision. I treat design and performance as constraints from day one, not bolt it on after.

Currently building SpicyDevs, and Posterium, a movie and TV poster platform backed by an API for metadata resolution, hosted compute, and large-scale data consolidation.
